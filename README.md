## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

```bash
git clone git@github.com:moise10r/kreatoors_api_test.git
cd ./kreatoors_api_test
```

### Install

```bash
npm install
```

### To Run it

```bash
npm run dev
```

### Build

```bash
npm run build
```
